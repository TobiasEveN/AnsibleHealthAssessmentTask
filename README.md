**Brief description**
This repository contains a Google Colab notebook and a Python script that parse a markdown Google Doc using the Google Docs API.

**Dependencies**
google-api-python-client google-auth-httplib2 google-auth-oauthlib

**How to run in Colab**
1. Open the `Assessmenttask.ipynb` in Google Colab.
2. Run cells top-to-bottom.
